# **App Name**: Élégance Portfolio

## Core Features:

- Homepage Project Showcase: Visually highlight key projects on the homepage in an appealing layout.
- About Page: Dedicated 'About Me' page accessible via the top menu with a profile image.
- Sortable Project Gallery: Projects can be sorted by category and software used.
- Visual Preview Upload: Functionality to upload preview files (images, videos) for each project.
- Elegant, Transparent Menu: A simple, elegant and transparent navigation menu.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a sophisticated, understated background.
- Background color: Light gray (#FAFAFA), slightly desaturated to maintain a clean and modern feel.
- Accent color: Dark blue (#2E3A87), analogous to beige but with stronger saturation, used sparingly for key interactive elements.
- Body font: 'Inter', a grotesque-style sans-serif, for clean and readable body text and headlines.
- Minimalist layout with ample whitespace to draw attention to showcased projects.
- Simple line icons for categories and software filters.
- Subtle fade-in animations when loading projects or navigating pages.
